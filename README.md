## Project Description
The SauceDemo (Swag Labs) website is a demo e-commerce platform provided by Sauce Labs, designed specifically for practicing software testing skills.
It simulates a real online shopping experience where users can log in, browse products, add items to the cart, and complete the checkout process. This makes it an ideal environment for both manual and automated testing.
Key Features
- Login system (valid & invalid scenarios)
-  Product listing and filtering
-  Add to cart / remove from cart
-  checkout process
-  Simple UI suitable for testing usability
  
 Managed a full project lifecycle using Jira and Zephyr Scale on the demo e-commerce Web https://www.saucedemo.com/.
- I set up the Scrum board, defined Epics & User Stories, wrote and executed Test Cases, tracked defects, and participated in release validation.
- This hands-on experience showed me how requirements → user stories → tests → defects → delivery all connect seamlessly.
